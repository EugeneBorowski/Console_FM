Консольный файловый менеджер.

Приложение позволяет просматривать каталоги, копировать удалять и выводить информацию о файлах и каталогах.
При первом запуске создается файл с настройками Console_FM.dll.config в котором записываются параметры по умолчанию.

**Список команд**

1. List
	Команда позволяет просматривать содержимое каталога и постранично выводить результат на экран, для постраничного вывода используйте клавиши Вверх и Вниз, для выхода нажмите клавишу ESC.
	По умолчанию, количество уровней отображения равняется 2, данную настройку можно изменять в файле настроек приложения в параметре paginglevel.
	Команда сохраняет последний введенный доступный каталог в файле настроек в параметр directory.
	Синтаксис команды должен быть в формате List <каталог>
2. Copy
	Команда позволяет копировать каталоги или файлы
	Если в каталога назначения не существует, он создается автоматически
	В случае совпадения име файла, файл автоматически заменяется
	Синтаксис команды должен быть в формате Copy <источник> <назначение>
3. Remove
	Команда позволяет удалять файлы либо каталоги.
	Удаление каталогов происходит с удалением всех подпапок и файлов.
	Синтаксис команды должен быть в формате Remove <каталог или файл> 
4. Info
	Команда позволяет отобразить информация о файле или каталоге
	Для файла это Имя, Размер в байтах и Информация об аттрибутах
	Для каталога это Имя и нформация об аттрибутах
	Синтаксис команды должен быть в формате Info <каталог или файл>
5. Exit
	Команда выхода из приложения.
